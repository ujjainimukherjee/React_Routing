# Installation #

## Server ##
This assumes at least node 10 is installed

```
cd server
```

#### Install ####
```
npm install
```

#### Run back-end tests ####
```
npm run test                  # alternatively, npm run test:watch for watch mode
```

#### Run the API ####
```
npm start
```

## Client ##
```
cd client
```

#### Install ####
```
npm install
```

#### Run front-end tests ####
```
npm test
```

#### Run the client ####
```
npm start
```
